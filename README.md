# Задание 1.
Декомпозиция производится с использованием Module Federation
Module Federation и Single-SPA предоставляют схожие возможности, однако настройка Single-SPA более комплексная (помимо webpack требуется настройка importMaps и root-config).
При этом преимущество в виде использования разных фреймворков в данном приложении не реализуется.
Плюс я уже познакомился с Module Federation в примере.

Микрофронтенды:
auth - модуль аутентификации
gallery - галерея с изображениями(добавление/удаление/просмотр/лайки)
profile - редактирование профиля пользователя
host - основное приложение
shared-library - разделяемая библиотека хранящая единый контекст пользователя
