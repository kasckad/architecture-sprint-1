# Проект: Микрофронтенды с использованием Module Federation

## Задание 1.1

Данный проект лучше разделить на микрофронтенды с использованием **Module Federation**. Это решение подходит, так как во всём проекте на фронтенде используется **React**, и компоненты будут иметь общие зависимости.

### Компоненты фронтенда

- **Host**  
  Контейнер для Module Federation, который объединяет все микрофронтенды.

- **Auth**  
  Отвечает за аутентификацию и регистрацию пользователей.  
  Компоненты: `Login.js`, `Register.js`.

- **Profile**  
  Отвечает за редактирование профиля пользователя.

- **Images**  
  Отвечает за загрузку и удаление фотографий.

- **Likes**  
  Отвечает за работу с лайками фотографий.

## Задание 1.2

Все микрофронтенды находятся в директории `frontend/microfrontend`. Далее эта директория считается текущей.

- **`likes/`**  
  Приложение, ответственное за обработку лайков.

- **`profile/`**  
  Приложение, ответственное за редактирование профиля пользователя.
