Задание 1
Для разбивки фронта на микрофронтенды выбран Module Federation так как все микрофронтенды реализованы на одном фреймворке React. Single SPA для данногоприложения избыточен.
Приложение разбито на 2 микрофронденда и общую  часть обеспечивающую общий стиль и функцоионал.
1. Центральное приложение. берет на себя функции Логина и Регистрации пользователя. Так же формирует отображение сквозных  элементов приложения: Шапку и футтер.
2. Микрофронтенд Gallery -  Все функции и элементы визулаизации связанные с карточками мест: Отображение карточек. добавление новых мест, загрузка фотографий, механизм лайков.
3. Микрофронтенд  User - Все функции и элементы визуализации связанные с пользователем: Аватар, данные о пользователе. Редактирование профиля.
